# Cinema Audio Boost

Landing page de pre-lanzamiento para `Cinema Audio Boost`, una extensión de Chrome enfocada en mejorar claridad de diálogos, volumen y sensación de sonido tipo cine en plataformas de streaming.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo local

```bash
npm install
npm run dev
```

Abre `http://localhost:3000` para ver la landing.

## Build de producción

```bash
npm run build
npm start
```

## Contenido incluido

- Landing bilingüe `ES / EN`
- Hero de pre-lanzamiento
- Beneficios, plataformas, presets y FAQ
- Captura de email solo UI
- Páginas reales de `privacy` y `support`

## Deploy

Está lista para desplegarse en Vercel como una app Next.js estándar.
